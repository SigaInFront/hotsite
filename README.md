# Siga in Front

> Hotsite para o evento de 2014

### Linguagens

- Node: NPM, Grunt e Bower
- Ruby: Sass

### Como utilizar

Baixe as dependências do NPM

```shell
npm install
```

Baixe as dependências do Bower

```shell
bower install
```

Ative o Grunt

```shell
grunt w
```
